📖 Panduan Pengguna Sistem Informasi Arsip Desa

Selamat datang di Sistem Informasi Arsip Digital Desa. Aplikasi ini dirancang untuk memudahkan pengelolaan, penyimpanan, dan pencarian dokumen arsip desa dengan tampilan yang modern dan mudah digunakan.

🚀 Memulai Aplikasi

1. Login Administrator

Halaman pertama yang akan Anda temui adalah halaman Login. Keamanan adalah prioritas, sehingga hanya admin yang terdaftar yang dapat mengakses sistem.

  • Username Default: admin
  • Password Default: admin123
  • Instruksi: Masukkan username dan password pada kolom yang tersedia, lalu klik tombol Masuk ke Sistem.

📊 Dashboard Utama

Setelah berhasil login, Anda akan diarahkan ke Dashboard. Ini adalah pusat kendali aplikasi.

Fitur Dashboard:

  1. Kartu Statistik: Di bagian atas, Anda dapat melihat ringkasan cepat:
    • Total Arsip: Jumlah seluruh dokumen yang tersimpan.
    • Per Kategori: Jumlah dokumen yang dikelompokkan berdasarkan jenisnya (Surat Masuk, Surat Keluar, dll).
  2. Pencarian & Filter:
    • Cari: Ketik judul atau deskripsi pada kolom pencarian untuk menemukan dokumen spesifik.
    • Filter Kategori: Pilih kategori dari dropdown untuk menampilkan jenis dokumen tertentu saja.
    • Tombol Reset: Muncul otomatis saat Anda sedang mencari/memfilter, klik untuk kembali melihat semua data.
  3. Tabel Arsip: Daftar dokumen ditampilkan dalam tabel yang rapi.
    • Sortir (Urutkan): Klik pada judul kolom No untuk mengurutkan data dari yang terbaru atau terlama.
    • Aksi: Tombol untuk Lihat/Unduh file, Edit data, atau Hapus arsip.
  4. Navigasi Halaman (Pagination): Jika data banyak, gunakan tombol Sebelumnya dan Selanjutnya di bagian bawah untuk berpindah halaman.

📝 Mengelola Arsip

1. Menambah Arsip Baru

Klik tombol ➕ Tambah Baru di pojok kanan atas Dashboard.

  • Judul Arsip: Masukkan nama dokumen (Wajib).
  • Kategori: Pilih jenis dokumen (Surat Masuk, Dokumen Peraturan, dll).
  • Deskripsi: Tuliskan keterangan singkat.
    Catatan: Maksimal 50 kata. Sistem akan otomatis memotong jika lebih.
  • Upload File:
    • Klik area kotak bintik-bintik atau ikon awan ☁️ untuk memilih file dari komputer Anda.
    • Setelah file dipilih, nama file akan muncul sebagai konfirmasi.
  • Klik Simpan Arsip untuk menyimpan.

2. Mengedit Arsip

Pada tabel di Dashboard, klik tombol Edit di kolom Aksi. Anda dapat mengubah judul, kategori, deskripsi, atau mengganti file yang sudah ada dengan yang baru.

3. Menghapus Arsip

Klik tombol Hapus (warna merah) pada arsip yang ingin dihapus. Konfirmasi peringatan yang muncul untuk menghapus data secara permanen.

⚙️ Pengaturan Akun

Untuk keamanan, sangat disarankan mengubah password default.

  1. Klik menu ⚙️ Pengaturan di Sidebar (menu samping).
  2. Password Lama: Masukkan password Anda saat ini.
  3. Password Baru: Masukkan password baru yang diinginkan.
  4. Konfirmasi Password: Masukkan ulang password baru untuk memastikan tidak ada salah ketik.
  5. Klik Simpan Perubahan.

🚪 Keluar (Logout)

Jika telah selesai menggunakan aplikasi, klik menu 🚪 Logout di bagian bawah Sidebar untuk keluar dari sistem dengan aman.
